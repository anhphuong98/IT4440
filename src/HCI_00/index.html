Môt vài lưu ý nhỏ trong project của lớp

update lần cuối Phùng Luân lúc 10/09/2019

Hello mọi người, hôm nay mình sẽ viết bài này để lưu ý một số điều nhỏ trong project của lớp mình.

Bài post này sẽ được mình update mỗi khi có vấn đề chung mà nhiều bạn thắc mắc và mình sẽ gán đầu ở trang project của lớp để các bạn có thể theo dõi thường xuyên.

Vấn đề sử dụng git.

Thực ra ban đầu mình không biết đa số các bạn dùng github, bitbucket, gitlab... hay gì, nhưng vì cần phải tạo repo chung cho toàn lớp nên mình đã dùng github.

Về các sử dụng thì đa số các bạn chắc đều sử dụng qua rồi, nếu bạn nào mới sử dụng thì có thể xem hướng dẫn cơ bản TẠI ĐÂY là đủ để các bạn có thể sử dụng git trong porject này.

Các nhóm mình đã chia folder sẵn để tránh vấn đề conflict xảy ra nên các bạn sẽ chỉ chỉnh sửa code trong thư mục của nhóm, không chỉnh sửa bất kỳ dòng code nào ngoài phạm vi nhóm của bạn nhé. ^^

Mỗi nhóm sẽ tạo 1 branch riêng để phát triển modules của các bạn, tránh việc push loạn xì ngầu vào nhánh chính. Mình không chặn các bạn push vào master về cơ bản là do mình đã phân chia folder cho từng nhóm và mỗi nhóm chỉ 1 account git nên việc conflict là điều khó xảy ra ^^

Mỗi nhóm nên tạo 1 link github nhỏ để các bạn làm việc, sau đấy khi hoàn chỉnh một nhiệm vụ thì lại push lên repo của lớp. Như thế để dễ quản lý tiến độ của các thành viên trong nhóm với nhau. ^^

Các bạn nên update sản phẩm của mình lên git thường xuyên, để mình đẩy lên server xem kết quả. Vì khi làm trên local và trên server (ubuntu) môi trường khác nhau có thể sinh lỗi code thì còn biết lỗi mà fix sớm :v

Cách điều hướng trang trong project

Trong quá trình sử dụng có không ít nhóm có nhiều trang, hoặc cần click vào button này thì trỏ đến trang của nhóm khác vân vân và mây mây... Ở thanh menu của chính trang này mình đã làm ví dụ về việc trỏ link trong nhóm và đến nhóm khác, các bạn có thể vào file html này trong source code để tham khảo nhé!!!.

Chú ý là các file html để hiện thị view của nhóm phải là con của folder nhóm bạn, y như file index.html mà mình tạo sẵn trong các nhóm nhé.

Lời kết thúc
